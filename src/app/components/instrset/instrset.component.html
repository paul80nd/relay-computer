ALU 8
1 0 0 0 r f f f
ALU Operations: Performs an arithmetic or logic operation on the B (and optionally C) register(s)
  r = destination register
      0-A 1-D
fff = function code
      000-NOP 001-ADD 010-INC       011-AND 100-OR  101-XOR       110-NOT 111-SHL
Note: SHL is a circular shift left (rotate left).
GOTO 24
1 1 d s c z n x
h h h h h h h h
l l l l l l l l
Branch/Call & 16-bit Load Immediate: Branches to a given address if stated condition register flag(s) is set. Address of next instruction can optionally be saved in XY register. M register can also be loaded with 16-bit value (without jump).
d = destination register
    0-M 1-J
s = jump if sign bit set
c = jump if carry bit set
z = jump if zero bit set
n = jump if zero bit not set
x = load XY=PC before jump

llllllll = address low byte
hhhhhhhh = address high byte
MOV-8 8
0 0 d d d s s s
8-Bit Move: Copies the content of one 8-bit register to another
ddd = destination register
      000-A  001-B  010-C 011-D
      100-M1 101-M2 110-X 111-Y
sss = source register
      000-A  001-B  010-C 011-D
      100-M1 101-M2 110-X 111-Y
MOV-16 10
1 0 1 0 0 d s s
16-Bit Move: Copies the content of one 16-bit register to another
 d = destination register
     0-XY 1-PC
ss = source register
     00-M 01-XY 10-J 11-AS
Note: AS takes the front panel switches as the low byte with zero as the high byte.
SETAB 8
0 1 r v v v v v
Load Immediate: Loads a value between -16 and +15 in register A or B
    r = destination register
        0-A 1-B
vvvvv = value (sign extended)
STORE 12
1 0 0 1 1 0 r r
Store Register: Stores the value in register A, B, C or D into the memory address pointed to by register M
rr = destination register
     00-A 01-B 10-C 11-D
HALT 10
1 0 1 0 1 1 1 r
Halt: Halts execution of the program (optionally reseting the program counter from the front panel switches)
r = reload from switches
    0-No 1-Yes
